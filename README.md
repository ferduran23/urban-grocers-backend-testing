# Urban Grocers Backend Testing

## Project Overview
Backend testing performed for a product delivery platform API.

## Project Achievements

- Created a checklist to validate a new backend API feature, ensuring all endpoints met functional and business requirements.

- Performed manual API testing using Postman by executing `GET`, `POST`, `PUT`, and `DELETE` requests with both valid and invalid data sets to verify:
  - Server responses (HTTP status codes)
  - Appropriate error messages
  - Required field validation
  - Data integrity within JSON responses

- Identified and documented bugs and unexpected behaviors in Jira, including detailed descriptions, reproduction steps, and supporting evidence such as screenshots and Postman exports.

- Validated key backend functionalities, including:
  - User authentication
  - Unauthorized access handling
  - Product inventory management through API endpoints
  - Data update logic such as stock and pricing updates

- Participated in a Scrum-based workflow within an Agile development environment.

**Date:** February 2026  
**Tools Used:** Postman, Jira

This project strengthened my backend testing skills by providing hands-on experience validating API functionality, business logic, and data integrity within a real-world application scenario.
